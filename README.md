# Mix2T Calc PRO ⛽🛢️

Calculadora de mistura de óleo para motores 2 tempos (Motosserra, Roçadeira, Mobilete, Barcos, etc). Um PWA (Progressive Web App) moderno, rápido e funcional, focado em alta performance e uso offline.

![PWA](https://img.shields.io/badge/PWA-Pronto-blue?style=for-the-badge)
![TypeScript](https://img.shields.io/badge/TypeScript-Vanilla-blue?style=for-the-badge)

## ✨ Funcionalidades PRO

- **Cálculo Inteligente**: Modo direto (Gasolina -> Óleo) e Modo Inverso (Óleo -> Gasolina).
- **Gestão de Equipamentos**: Salve perfis personalizados (Nome + Proporção) para carregar com um clique.
- **Histórico de Misturas**: Acesso rápido às últimas 10 misturas realizadas.
- **Calculadora de Custos**: Insira o preço do combustível e do óleo para saber o custo real da sua mistura.
- **Atalhos de Volume**: Botões rápidos para 1L, 5L, 10L e 20L.
- **Compartilhamento**: Envie os detalhes da mistura via WhatsApp instantaneamente.
- **Suporte Offline**: Funciona em qualquer lugar, mesmo sem sinal de internet (via Service Workers).
- **Design de Alta Legibilidade**: Temas Escuro e Claro otimizados para uso sob luz solar intensa.

## 🛠️ Tecnologias Utilizadas

- **Vite**: Build system ultra rápido.
- **TypeScript**: Robustez e segurança no código.
- **Vanilla CSS**: Estilização premium com variáveis CSS e animações fluidas.
- **IndexedDB (idb-keyval)**: Armazenamento local persistente e performático.
- **Lucide Icons**: Conjunto de ícones leves e elegantes.
- **Vite PWA Plugin**: Configuração de Service Workers para funcionamento offline.

## 🚀 Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/chwpym/calculadora-oleo-2t.git
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Para gerar a versão de produção:
```bash
npm run build
```

## 📱 Instalação como App

Este projeto é um PWA. Para instalar no seu celular:
1. Abra o link do projeto no Chrome (Android) ou Safari (iOS).
2. Clique no menu do navegador (três pontos ou compartilhar).
3. Selecione **"Instalar aplicativo"** ou **"Adicionar à tela de início"**.
